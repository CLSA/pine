HTML Syntax

All HTML elements which enclose text must have an open and close element. Elements which do not enclose text don't need to be closed.

For example

  • This is incorrect: <p>This is a paragraph. (no closing tag)
  • This is also incorrect: <b>This is a paragraph.<b> (closing tag is missing slash "/")
  • This is correct: <b>This is a paragraph.</b>
